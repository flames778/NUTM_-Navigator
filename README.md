# NUTM Navigator Pro

**NUTM Navigator Pro** is a premium campus navigation and social AI web application designed for the NUTM community. It provides advanced campus navigation, community features, and AI-powered assistance for students, staff, and guests.

## Features

- **Campus Navigation:** Search for locations, get routes, and explore the campus map interactively.
- **Authentication:** Student login, registration, admin access, and guest/anonymous login.
- **Community Feed:** Post updates, interact with other users, and stay informed about campus events.
- **AI Assistance:** Ask questions about campus locations and get instant answers.
- **Modern UI:** Responsive, theme-switching (dark/light), and mobile-friendly design.
- **Admin Panel:** Manage users and view campus statistics.

## Technologies Used

- **HTML5, CSS3, JavaScript (Vanilla)**
- **Firebase:** Authentication and Firestore database
- **Leaflet.js:** Interactive campus maps
- **Font Awesome:** Icons
- **XLSX.js:** Excel file support

## Getting Started

1. Clone or download this repository.
2. Open `complete_set.html` in your browser.
3. Configure your Firebase project and update the Firebase config in the HTML file if deploying for production.

## Usage

- **Students:** Register or log in with your NUTM email.
- **Admins:** Use the provided admin credentials for management access.
- **Guests:** Explore basic features without registration.
